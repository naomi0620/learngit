##8.”==”和“===”和Object.is()的不同 ##
*== 只比较值不比较类型
*=== 不仅比较值也比较类型 NaN===NaN false +0===-0 true 
*Object.is() 值与类型都比较 NaN===NaN true +0===-0 false
